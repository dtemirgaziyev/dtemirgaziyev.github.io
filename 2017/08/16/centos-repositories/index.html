Подключение сторонних репозиториев в CentOS

centos repository yum

Подключение сторонних репозиториев в CentOS (RHEL) и защита основного репозитория с помощью плагина yum-priorities.

Защита основного репозитория

Перед подключением сторонних репозиториев рекомендуется защитить основной репозиторий посредством настройки плагина yum-priorities.

Установите плагин для настройки приоритетов:

sudo yum -y install yum-plugin-priorities

Установите приоритет равный 1 в конфигурационных файлах основного репозитория:

sudo sed -i '/name=*/a priority=1' /etc/yum.repos.d/CentOS-*.repo

Дополнительная информация о yum-priorities доступна на Wiki.

EPEL

Подключение репозитория EPEL:

sudo yum install epel-release

Nux Dextop

Подключение репозитория Nux Dextop:

sudo yum localinstall http://li.nux.ro/download/nux/dextop/el7/x86_64/nux-dextop-release-0-5.el7.nux.noarch.rpm

ELRepo

Подключение репозитория ELRepo:

sudo yum localinstall http://www.elrepo.org/elrepo-release-7.0-3.el7.elrepo.noarch.rpm

Remi

Подключение репозитория Remi:

sudo yum localinstall https://rpms.remirepo.net/enterprise/remi-release-7.rpm

IUS

Подключение репозитория IUS:

sudo yum localinstall https://centos7.iuscommunity.org/ius-release.rpm
