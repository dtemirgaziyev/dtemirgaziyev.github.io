Fedora - обновление системы

fedora system upgrade dnf

Процедура обновления Fedora на следующую версию дистрибутива. В данной заметке обновление происходит с версии 25 на 26.

Рекомендуемый способ

Обновление Fedora c помощью специального плагина system-upgrade. Этот способ является официально рекомендуемым способом обновления начиная с Fedora 21.

Установим специальный плагин для обновления системы:

sudo dnf -y install dnf-plugin-system-upgrade

Обновим все установленные пакеты в системе:

sudo dnf upgrade --refresh

Скачиваем пакеты для обновления системы, где --releasever=26 - версия на которую хотим обновиться:

sudo dnf system-upgrade download --releasever=26

Запустим процесс обновления системы после перезагрузки:

sudo dnf system-upgrade reboot

Альтернативный способ

Рекомендуется перейти в виртуальную консоль (Ctrl+Alt+F2) или перейти в multi-user уровень выполнения:

sudo systemctl isolate multi-user.target

Обновим все установленные пакеты в системе:

sudo dnf upgrade --refresh

Очистим кэш в пакетном менеджере:

sudo dnf clean all

Обновляем дистрибутив до версии 26:

sudo dnf --releasever=26 distro-sync

Перезагружаем систему:

systemctl reboot
