Создание загрузочного USB накопителя в Linux

dd bootable usb

Как создать загрузочный USB накопитель в любом дистрибутиве Linux.

Создать загрузочный USB накопитель можно с помощью dd:

sudo dd if=~/Fedora-Workstation-Live-x86_64-25-1.3.iso of=/dev/sdb bs=8M status=progress

Пояснение:

  • if=~/Fedora-Workstation-Live-x86_64-25-1.3.iso - указываем правильный путь к ISO образу;
  • of=/dev/sdb - указываем правильный путь к USB устройству;
  • bs=8M - записывать блоками по 8 Мбайт.

Для любителей GUI-ёв есть такие приложения как GNOME Disks и Fedora Media Writer.

sudo dnf -y install gnome-disk-utility
sudo dnf -y install mediawriter
