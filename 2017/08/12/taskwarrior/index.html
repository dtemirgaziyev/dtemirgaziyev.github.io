Taskwarrior

taskwarrior fedora centos

Taskwarrior - гибкая и мощная консольная утилита для управления списком задач из командной строки.

Установка Taskwarrior в Fedora:

sudo dnf -y install task

Для установки Taskwarrior в CentOS (и RHEL) требуется подключить репозиторий EPEL:

sudo yum install epel-release
sudo yum -y install task

После первого запуска Taskwarrior вам будет предложено создать конфигурационный файл: ~/.taskrc, введите «yes» чтобы согласится.

task

Добавляем свою первую задачу в Taskwarrior:

task add My first task

Смотрим добавленные задачи в списке задач:

task list

Отмечаем что начали работать над выполнением первой задачи:

task 1 start

Изменяем приоритет на самый высокий (H) для первой задачи:

task 1 modify priority:H

Изменяем проект для первой задачи:

task 1 modify project:Myproject

Отмечаем что завершили работу над первой задачей:

task 1 done

Для дальнейшего изучения возможностей утилиты Taskwarrior читайте официальную документацию.
