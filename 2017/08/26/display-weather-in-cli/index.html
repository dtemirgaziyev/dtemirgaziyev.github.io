Посмотреть прогноз погоды в командной строке

Posted on Sat, Aug 26, 2017 | #curl #wttr.in

Узнаем прогноз погоды в командной строке с помощью сервиса wttr.in и утилиты curl.

Посмотреть прогноз погоды в командной строке:

curl wttr.in/Astana
curl wttr.in/Almaty

Посмотреть справку:

curl wttr.in/:help

Репозиторий wttr.in на GitHub.
