Удаление старых ядер в CentOS

Posted on Sun, Aug 27, 2017 | #CentOS #yum #package-cleanup #sed

Корректное удаление старых ядер и очистка места на разделе /boot в CentOS (и RHEL).

Установим пакет с дополнительными утилитами для yum:

sudo yum -y install yum-utils

Удаляем старые ядра, где --count=2 - количество ядер которое нужно оставить:

sudo package-cleanup --oldkernels --count=2

Для того что бы пакетный менеджер сам контролировал количество установленных в системе ядер нужно изменить параметр installonly_limit в конфигурационном файле /etc/yum.conf:

sudo sed -i 's/installonly_limit=5/installonly_limit=2/' /etc/yum.conf
