Подключение сторонних репозиториев в Fedora

Posted on Thu, Aug 10, 2017 | #fedora #dnf #repository

В данной заметке вы узнаете какие полезные сторонние репозитории доступны для подключения в Fedora.

RPMFusion

Подключение и отключение репозитория RPMFusion (free, nonfree):

sudo dnf install https://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm https://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm
sudo dnf config-manager --set-disabled rpmfusion-free rpmfusion-free-updates rpmfusion-nonfree rpmfusion-nonfree-updates

Russian Fedora

Подключение и отключение репозитория Russian Fedora (free, nonfree, fixes):

sudo dnf install http://mirror.yandex.ru/fedora/russianfedora/russianfedora/free/fedora/russianfedora-free-release-stable.noarch.rpm http://mirror.yandex.ru/fedora/russianfedora/russianfedora/nonfree/fedora/russianfedora-nonfree-release-stable.noarch.rpm http://mirror.yandex.ru/fedora/russianfedora/russianfedora/fixes/fedora/russianfedora-fixes-release-stable.noarch.rpm
sudo dnf config-manager --set-disabled russianfedora-free russianfedora-free-updates russianfedora-nonfree russianfedora-nonfree-updates russianfedora-fixes russianfedora-fixes-updates

Negativo17

Подключение и отключение репозитория Negativo17:

sudo dnf config-manager --add-repo=http://negativo17.org/repos/fedora-negativo17.repo
sudo sed -i 's/enabled=1/enabled=0/' /etc/yum.repos.d/fedora-negativo17.repo

Bumblebee

Подключение и отключение репозитория Bumblebee:

sudo dnf install http://install.linux.ncsu.edu/pub/yum/itecs/public/bumblebee/fedora$(rpm -E %fedora)/noarch/bumblebee-release-1.2-1.noarch.rpm http://install.linux.ncsu.edu/pub/yum/itecs/public/bumblebee-nonfree/fedora$(rpm -E %fedora)/noarch/bumblebee-nonfree-release-1.2-1.noarch.rpm
sudo dnf config-manager --set-disabled bumblebee bumblebee-nonfree
