Использование репозитория CR в CentOS

Posted on Mon, Aug 28, 2017 | #centos #repository #cr

Репозиторий CR (Continuous Release) позволяет устанавливать исправления и обновления безопасности которые уже были выпущены для новой версии RHEL.

На момент написания заметки, текущей версией CentOS является 7.3 тогда как текущая версия RHEL уже 7.4. Здесь и нужен репозиторий CR который позволит установить исправления и обновления безопасности вышедших для RHEL 7.4 не дожидаясь выхода CentOS 7.4.

Подключение репозитория CR в CentOS 7:

sudo yum update
sudo yum-config-manager --enable cr

Подключение репозитория CR в CentOS 5 и 6:

sudo yum install centos-release-cr

Подробную информацию про репозиторий CR можете почитать на Wiki.
