Настройка истории команд в Bash

bash history

Несколько советов по настройке истории команд в командной оболочке Bash.

Откройте файл ~/.bashrc в текстовом редакторе:

vim ~/.bashrc

Добавьте в конец файла следующие строки:

export HISTTIMEFORMAT='%h %d %H:%M:%S '
export HISTSIZE=5000
export HISTFILESIZE=5000
export HISTCONTROL="ignoreboth"
export HISTIGNORE="history*:pwd*:ls*"
PROMPT_COMMAND='history -a'

Пояснение:

  • export HISTTIMEFORMAT=’%h %d %H:%M:%S ‘ - показывать дату и время в истории команд;
  • export HISTSIZE=5000 - увеличить размер истории команд до 5000 команд;
  • export HISTFILESIZE=5000 - увеличить размер истории команд до 5000 строк;
  • export HISTCONTROL="ignoreboth" - не сохранять в истории, команды начинающиеся с пробела и повторяющиеся команды;
  • export HISTIGNORE="history*:pwd*:ls*" - не сохранять в истории, команды указанные в этом списке;
  • PROMPT_COMMAND=’history -a’ - сохранять историю команд сразу а не после завершения сеанса.

Примените сделанные изменения:

source ~/.bashrc
