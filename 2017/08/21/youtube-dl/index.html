youtube-dl

Posted on Mon, Aug 21, 2017 | #fedora #centos #youtube-dl

youtube-dl - консольная утилита для загрузки видео с YouTube и множества других сайтов и видеохостингов.

Установить в Fedora:

sudo dnf -y install youtube-dl

В CentOS для установки нужно подключить репозиторий EPEL:

sudo yum install epel-release
sudo yum -y install epel-release

Чтобы скачать видео достаточно команды:

youtube-dl https://ссылка_на_видео

Смотрим список с доступными форматами и разрешениями для видео:

youtube-dl -F https://ссылка_на_видео

Скачиваем видео в выбранном формате и разрешении, для этого укажите код (в моем случае - 22):

youtube-dl -f 22 https://ссылка_на_видео
