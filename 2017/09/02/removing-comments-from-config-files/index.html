Очистка конфигурационных файлов от комментариев

Posted on Sat, Sep 2, 2017 | #grep #sed #bash functions

Как легко очистить конфигурационный файл от комментариев с помощью стандартных утилит grep и sed.

Вывести содержимое конфигурационного файла без комментариев с помощью утилит grep и sed:

grep -Ev '(^[[:space:]]*([#;!].*)?$)' /etc/httpd/conf/httpd.conf
sed 's/#.*//;/^\s*$/d' /etc/httpd/conf/httpd.conf

Создать на основе оригинального конфигурационного файла, новый очищенный от комментариев конфигурационный файл можно так:

cp /etc/httpd/conf/httpd.conf /etc/httpd/conf/httpd.conf.orig
grep -Ev '(^[[:space:]]*([#;!].*)?$)' /etc/httpd/conf/httpd.conf.orig > /etc/httpd/conf/httpd.conf

Можно добавить функцию catcfg() в файл ~/.bashrc при необходимости частого использования:

catcfg() {
grep -Ev '(^[[:space:]]*([#;!].*)?$)' $1
}

Пример использования:

catcfg /etc/httpd/conf/httpd.conf
