Уменьшение заголовков окон в GNOME 3

Posted on Wed, Oct 4, 2017 | #gnome

Короткая заметка про уменьшение заголовков окон в GNOME 3, не знаю зачем их сделали такими большими.

Создаём файл ~/.config/gtk-3.0/gtk.css и добавляем следующее содержимое:

headerbar.default-decoration {
 padding-top: 3px;
 padding-bottom: 3px;
 min-height: 0px;
 font-size: 1em;
}

headerbar.default-decoration button.titlebutton {
 padding: 0px;
 min-height: 0px;
}
